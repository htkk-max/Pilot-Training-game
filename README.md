# 飞行员训练游戏

## 游戏简介

飞行员训练（Pilot Training）是一款有趣的网页飞行游戏。玩家扮演飞行员，通过完成各种飞行任务来提升自己的飞行等级，解锁新的飞机和地图。

## 游戏特色

- **多种任务类型**：避开障碍物、收集物品、穿越圆环等
- **飞机选择**：多种飞机可供选择，各有特色
- **地图选择**：多个不同风格的地图场景
- **等级系统**：通过完成任务提升飞行员等级
- **星星收集**：收集星星用于解锁新内容
- **任务系统**：多样化的飞行任务挑战

## 游戏玩法

### 基本操作

**PC端操作：**
- 鼠标点击屏幕左侧或右侧控制飞机左右移动
- 使用键盘左右方向键控制飞机移动

**移动端操作：**
- 点击屏幕左侧或右侧控制飞机左右移动

### 任务类型

1. **UFO 闪避**：躲避飞来的 UFO，尽可能存活更长时间
2. **树木闪避**：避开树木障碍，在绿色环境中生存 60 秒
3. **热气球闪避**：避开缓慢移动的热气球，生存 60 秒
4. **船只闪避**：避开向左移动的热气球，生存 60 秒
5. **圆环穿越**：穿过指定数量的圆环，沿着正确路径飞行
6. **燃料收集**：在 1 分钟内收集指定数量的燃料滴
7. **星星收集**：收集指定数量的闪亮星星

### 游戏目标

- 完成各项飞行任务
- 提升飞行员等级
- 解锁更多飞机和地图
- 挑战更高分数

## 技术特点

- **响应式设计**：完美支持桌面端和移动端设备
- **多种输入方式**：支持鼠标点击、键盘输入和触摸操作
- **Canvas 渲染**：使用 HTML5 Canvas 技术实现流畅的游戏画面
- **跨浏览器兼容**：支持主流现代浏览器

## 如何开始

1. 直接在浏览器中打开 `index.html` 文件
2. 点击开始按钮进入游戏
3. 选择飞机和地图
4. 按照任务提示完成飞行挑战
5. 享受飞行乐趣！

## 开发说明

本游戏使用纯网页技术开发，基于 HTML5 Canvas、JavaScript 和 CSS3 构建，无需安装任何插件或框架，直接在浏览器中运行即可。
